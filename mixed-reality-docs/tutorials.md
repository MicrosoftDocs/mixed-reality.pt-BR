# <a name="tutorials-and-sample-apps"></a>Aplicativos de exemplo e tutoriais

Atualmente, todos os nossos tutoriais e aplicativos de exemplo são Unity com base.  Como podemos desenvolver conteúdo para outras plataformas e mecanismos, você encontrará-los sob o título relevante no sumário.

## <a name="mr-learning-modules"></a>Módulos de aprendizado MR

Os módulos de aprendizado MR são projetados para ajudar os desenvolvedores a aprender técnicas e práticas recomendadas para o desenvolvimento de aplicativos de realidade misturada.  Os módulos inicias são para 2 HoloLens e implementam o Toolkit realidade mista 2.0 (MRTK 2.0).
* O módulo básico destina-se para mostrar como configurar seu projeto com MRTK, bem como implementar a interface do usuário, o mapeamento espacial e interações básicas e avançadas de desenvolvedores

## <a name="open-source-sample-apps"></a>Aplicativos de exemplo de código-fonte aberto

Os aplicativos de exemplo de código-fonte aberto são aplicativos que foram criados por nossas equipes e fornecemos o código-fonte para que os desenvolvedores pudessem examiná-la e aprenda com nossos projetos.

## <a name="mixed-reality-academy-tutorials"></a>Tutoriais da academia de realidade mista

Academia de realidade mista é um conjunto de tutoriais passo a passo online com os arquivos de projeto correspondente: 
* Os tutoriais abordam 100, 200 e 300 tópicos de nível, em que: Noções básicas sobre o projeto de nível 100 bastidores, MR nível 200 bastidores principais blocos de construção e integração de serviços de nuvem tampas de nível 300.
* A maioria dos cursos abordam conceitos aplicáveis ao HoloLens e fones imersivos em exposição (VR). 
* Cada tutorial é organizado por capítulo e incluir mais demonstrações em vídeo dos principais conceitos. 
* Um computador Windows 10 com o nome correto [as ferramentas instaladas](install-the-tools.md) é um pré-requisito comuns para concluir cada tutorial.

>[!NOTE]
>Os tutoriais da academia de realidade mista foram criados com o HoloLens (1º gen) e misto realidade fones Imersivos em exposição em mente.  Como tal, achamos que é importante deixar esses tutoriais em vigor para os desenvolvedores que ainda estiver procurando por orientação no desenvolvimento para esses dispositivos.  Esses tutoriais serão **_não_** ser atualizados com os conjuntos de ferramentas ou interações que está sendo usadas para o HoloLens 2 a mais recente.  Eles serão mantidos para continuar trabalhando nos dispositivos com suporte. Haverá uma nova série de tutoriais que serão lançados no futuro e que demonstra como desenvolver para o HoloLens 2.  Este aviso será atualizado com um link para esses tutoriais quando são lançadas.
